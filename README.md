Wheater application to send you a message every morning saying if it's raining or not in your current location.

1. You need to set up you precise location inside the "paramaters" dictionationary, using your latitude and your longitude.

2. It's set up to run every morning at 8 am UTC. Be free to change to set it to a better time for you.

3. To make it work, you need to create a twilio account, there's a tutorial there to set it all up, even for free if want.
